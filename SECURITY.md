# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Ethco AI, please report it to us privately.

**Please do not open a public issue.**

### How to Report

Email: hailegetaneh1221@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response Time

We aim to respond to security reports within 48 hours and will keep you updated on the progress.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Security Best Practices

- Never commit API keys or secrets
- Use environment variables for sensitive data
- Keep dependencies up to date
- Follow secure coding practices
