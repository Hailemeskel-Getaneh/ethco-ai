# Ethco AI Mobile App

This is the mobile application for the Ethco AI platform, built with React Native and Expo.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the app:
    ```bash
    npx expo start
    ```

## Development

-   **UI Components**: Located in `src/components/ui`.
-   **Screens**: Located in `src/app`.
-   **Theme**: Theme definitions in `src/theme`.

## key Features

-   AI-powered interactions
-   Modern, responsive UI
-   Cross-platform support (iOS, Android)
